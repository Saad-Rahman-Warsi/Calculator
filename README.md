# 📱 Android Calculator App

A sleek, lightweight, and native Android calculator application designed for quick, everyday arithmetic calculations with an intuitive material user interface.

---

## 🚀 Features

- **Core Math Engine:** Handles fluid addition, subtraction, multiplication, and division operations.
- **Dynamic Display:** Live input calculation rendering with overflow handling for long numbers.
- **Modern UI Styling:** Built adhering to native Android Design Guidelines for tactile feedback and clean layouts.
- **State Preservation:** Maintains calculation inputs across device configurations (such as screen rotations).
- **Error Trapping:** Active fail-safes preventing runtime application crashes from invalid inputs (e.g., division by zero, duplicate decimals).

---

## 🛠️ Built With

This application leverages standard native Android development tools to ensure high performance and low resource consumption:

- **Kotlin / Java** – Core backend logic, event-driven click listener handling, and math string processing.
- **XML Layouts** – Declarative user interface design utilizing `ConstraintLayout` or `GridLayout` for cross-screen adaptation.
- **Android SDK** – Compatible across a wide cross-section of Android API levels to ensure broad device support.
- **Gradle** – Build automation toolkit managing compilation assets, packaging, and dependency injection.
